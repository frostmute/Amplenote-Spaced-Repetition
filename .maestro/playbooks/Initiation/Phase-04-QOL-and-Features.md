# Phase 04: QOL Improvements and Feature Additions

This phase adds polish and user-centric features to make the Spaced Repetition experience delightful. We will add Quality of Life (QOL) enhancements that set the plugin apart and make it feel like a native Amplenote feature.

## Tasks

- [x] Brainstorm and document potential QOL features:
  - Created `docs/research/feature-ideas.md` with YAML front matter (`type: research`, `tags: [features, qol]`).
  - Outlined 4 potential QOL features including Cloze deletions, customizable review intervals, visually distinct formatting for due cards, and a review statistics dashboard.

- [ ] Implement improved flashcard formatting:
  - Modify the output generated by the plugin to use Amplenote's markdown features (quotes, highlights, or tables) to make questions and answers visually distinct and easy to read.
  - Add logic to automatically apply a `#srs/due` tag to notes that contain cards ready for review today.

- [ ] Add a review summary or stats generation feature:
  - Implement a function that calculates how many cards are due, how many were reviewed today, and the user's current retention rate.
  - Output this summary neatly at the top of a designated "Spaced Repetition Dashboard" note or via a notification alert.

- [ ] Update documentation to reflect new features:
  - Ensure any new markdown syntax or commands required by the user are documented within the plugin's metadata string or a dedicated help function.
